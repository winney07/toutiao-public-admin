#### main.js：入口文件

```
// 创建Vue根实例
// 把 router配置到根实例中
// 通过render 方法把App根组件渲染到#app入口节点
new Vue({
	router,
	render : h => h(App)
	// el: '#app' // 等价于$mount('#app')
}).$mount('#app')
```
